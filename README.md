# GEXORA

Modern ve responsive bir oyun keşif sitesi. Kullanıcıların oyun kategorilerini keşfetmesini, popüler oyunları incelemesini ve yeni oyunlar bulmasını sağlayan bir arayüz projesidir.

## 🌐 Canlı Demo

https://furkan4778.github.io/gexora

## 🛠️ Kullanılan Teknolojiler

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Responsive Design
* Font Awesome

## 📌 Özellikler

* Modern ve kullanıcı dostu arayüz
* Tam responsive tasarım
* Oyun kategorileri bölümü
* Trend oyunlar bölümü
* Oyun keşif kartları
* Hover animasyonları ve geçiş efektleri
* Mobil cihaz desteği

Bu proje, HTML ve CSS becerilerimi geliştirmek, responsive tasarım pratiği yapmak ve modern bir oyun platformu arayüzü oluşturmak amacıyla geliştirilmiştir.

## ⚠️ Not

Bu proje eğitim ve portföy amacıyla geliştirilmiştir ve aktif olarak güncellenmektedir.
